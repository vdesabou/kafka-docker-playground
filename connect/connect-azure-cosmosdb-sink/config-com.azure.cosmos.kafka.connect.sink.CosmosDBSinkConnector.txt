ID Strategy Parameters
==========================
🔘 id.strategy

A strategy used to populate the document with an ``id``. Valid strategies are: ``TemplateStrategy``, ``FullKeyStrategy``, ``KafkaMetadataStrategy``, ``ProvidedInKeyStrategy``, ``ProvidedInValueStrategy``. For each strategy, the full name of the strategy must be specified, e.g. com.azure.cosmos.kafka.connect.sink.id.strategy.TemplateStrategy. Configuration properties prefixed with``id.strategy`` are passed through to the strategy. For example, when using``id.strategy=com.azure.cosmos.kafka.connect.sink.id.strategy.TemplateStrategy`` , the property ``id.strategy.template`` is passed through to the template strategy and used to specify the template string to be used in constructing the ``id``.

	 - Type: CLASS
	 - Default: com.azure.cosmos.kafka.connect.sink.id.strategy.ProvidedInValueStrategy
	 - Importance: HIGH
	 - Required: false
